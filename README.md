# Task 1: Data Cleaning and Preprocessing

## Objective
Clean and prepare a raw dataset with missing values, duplicates, and inconsistent formats.

## Tools Used
- Python
- Pandas
- VS Code

## Dataset
Medical Appointment No Shows Dataset

## Steps Performed
- Checked missing values using isnull()
- Removed duplicate rows
- Renamed column headers to lowercase
- Standardized gender values
- Converted date columns to datetime
- Fixed age data type
- Removed negative ages

## Output
cleaned_medical_appointments.csv
